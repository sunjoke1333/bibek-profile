# Bibek Portfolio

A premium, high-performance personal portfolio built with Next.js, Framer Motion, and Tailwind CSS. 

## Overview

This portfolio is a showcase of my journey as a Solution Architect, Sales Leader, and Technical Engineer. It blends strategic professional milestones with a high-end digital aesthetic, optimized for performance and impact.

## Tech Stack

- **Framework**: Next.js
- **Styling**: Tailwind CSS (v4)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Components**: Shadcn/ui

## Key Sections

- **Identity**: A vibrant home base highlighting the core philosophy.
- **Milestones**: A data-driven professional timeline with quantifiable achievements.
- **Projects**: Deep-dive case studies focusing on problem-solution-impact narratives.
- **Bento Grid**: A unified digital identity hub linking all professional platforms.

## Development

```bash
npm install
npm run dev
```
